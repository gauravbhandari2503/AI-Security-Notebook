# AI Guidelines Directory

This folder contains guidelines and instruction sets intended for AI assistants and human developers working in this repository:

- **`AI_CODING_RULES.md`**: The primary rule set for any AI assistant (like Copilot, Claude, or Cursor) generating or reviewing code in this repository. It dictates security practices, strict typing, data privacy constraints, and secrets management when writing production code.
- **`building-ai-features/`**: Contains in-depth architectural and operational guidelines specifically for **building AI-integrated capabilities** within the application (e.g., handling LLM rate limits, output validation of LLMs, RAG patterns, logging AI usage, etc.). These should be referenced when the team is actively developing an AI-powered product feature.
